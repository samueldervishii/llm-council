FAQ

Short answers to the first questions people ask

What file types can I upload?

Cortex supports PDF, DOCX, TXT, Markdown, and CSV uploads for source analysis and file-aware conversations.

Are chats saved?

Yes. Sessions are stored in your account so you can return to them, search them, pin them, share them, or delete them later.

Can I export my work?

Yes. Sessions can be exported as JSON or Markdown, and document outputs can be exported as DOCX.

Can I control what gets shared?

Yes. Shared sessions are optional and only visible through the generated share link while sharing is enabled.
